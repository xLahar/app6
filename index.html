Thunder Vortex Ride Control System

Velocity Kingdom | Real-Time Systems Project

Thunder Vortex is a simulated theme park ride control system built on an ESP32 using FreeRTOS. The project monitors ride position, controls magnetic brakes, and responds to emergency stop events while demonstrating hard and soft real-time scheduling.

Demo Video

Project Features

  • ESP32 + FreeRTOS task scheduling
  • Emergency stop interrupt handling
  • Brake control and LED indicators
  • Sensor-based ride position monitoring
  • Wokwi simulation demo
